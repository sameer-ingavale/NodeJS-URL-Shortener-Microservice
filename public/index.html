Sameer's URL Shortener Microservice

User Stories

  • I can enter a URL in the input field and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example Usage

"https://sameers-url-shortener.herokuapp.com/YCoy9ID1

Example Output

{ "IP": "72.198.91.159", "Language": "en-US", "OS": "Mac OS 10.14.6" }
